

# LEARNING APP

## what and why
**Wanted a good learning app for btech students, as were students aswell**

## Proposed Features

1. **PYQ with Answer Key**
2. **Skill gap Analysis**
3. **Extra Curricular Skill Development**
4. **Placement Prep and Training**
5. **Proper Learning Environment**
6. **Interractive Lessions / Video Lessons / Audiobooks / MicroLessons / FlashCards / gamified learning etc**
7. **Active Learning Roadmap**
8. **Incentiviced Games (earn points to play games etc)**
9. **Proper Study Methods**


# Client Side Modules

## Module 1 : User Authentication and onboarding

* **Fetch UI from Server using API**
* **To be consisted of several pages**
* **Should Contain Firebase/Github logins, get ID token from providers, and create user if not exist otherwise login, PASETO TOKENS**
* **Cross platform UI**
* **User Onboarding** 
* *USER DATA TO BE STORED IN OUR OWN SERVERS NOT IN GOOGLE SERVERS*

## Module 2 : App Core
1. **SDUI Engine** : parse incoming json into widgets, handle actions
2. **Network Intercepting** : Attach PASETO token on every request, validate keys etc
3. **Offline Caching** : Store Json Payload of every visited screens, use cacehed data when there is no internet
